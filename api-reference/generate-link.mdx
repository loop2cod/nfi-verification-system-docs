---
title: "Generate Verification Link"
api: "POST https://api.nfi-clear.com/v1/generate-link"
description: "Create a new verification session and get a hosted link for your user."
---

## Body

<ParamField body="type" type="string" required>
  The type of verification flow. Must be `KYC_INDIVIDUAL` or `KYB_CORPORATE`.
</ParamField>

<ParamField body="metadata" type="object">
  Optional key-value pairs to store custom data (e.g., `userId`, `email`) associated with this verification.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful.
</ResponseField>

<ResponseField name="verificationId" type="string">
  The unique 16-character hex ID for the session.
</ResponseField>

<ResponseField name="url" type="string">
  The fully constructed URL to redirect your user to (e.g., `https://nfi-clear.com/verify/...`).
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.nfi-clear.com/v1/generate-link \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "KYC_INDIVIDUAL",
    "metadata": {
      "userId": "user_123"
    }
  }'
```

</RequestExample>
