---
title: "Generate Link"
api: "POST https://api.hux.com/api/v1/external/generate-link"
description: "Create a new verification session and get a unique link for your user."
---

### Body

<ParamField body="type" type="string" required>
  The type of verification to perform. derived from `KYC_INDIVIDUAL` or `KYB_CORPORATE`.
</ParamField>

<ParamField body="metadata" type="object">
  Key-value pairs of additional information you want to store with this verification (e.g., your internal user ID).
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the call was successful.
</ResponseField>

<ResponseField name="verificationId" type="string">
  The unique ID for this verification session.
</ResponseField>

<ResponseField name="url" type="string">
  The URL you should redirect your user to.
</ResponseField>

<RequestExample>
```bash Example Request
curl --request POST \
  --url https://api.hux.com/api/v1/external/generate-link \
  --header 'Content-Type: application/json' \
  --header 'x-api-key: <api-key>' \
  --data '{
  "type": "KYC_INDIVIDUAL",
  "metadata": {
    "userId": "user_123"
  }
}'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "verificationId": "a1b2c3d4e5f6",
  "url": "https://verify.hux.com/verify/a1b2c3d4e5f6"
}
```
</ResponseExample>
